#TaxxFlow Ai 

A high-conversion tax and bookkeeping web app built for SmartBooks by ReFURRM.  
This project combines a professional client-facing site, admin dashboard, and multi-step client intake flow with structured logging for AI-assisted analysis.

---

## Overview

TaxxFlow Ai is designed to feel like a high-end financial firm:

- Light-mode, accessibility-conscious UI
- Professional Emerald Green and Deep Navy color palette
- Clear funnels for leads, onboarding, and ongoing subscription services
- Admin tools for monitoring leads, subscriptions, and payments
- Structured logging to make AI-based debugging and analysis easier

---

## Key Features

### 1. Marketing & Client-Facing Site

**Home / Landing Page**

- Hero section with value props and clear “Get Started” CTAs
- Trust badges (IRS Authorized, SSL, SOC 2, PTIN, RON, volume of returns filed)
- Service grid for:
  - Bookkeeping
  - Corporate Tax Prep
  - IRS Resolution
  - RON Services
  - General Notary
  - Remote Returns
  - Pricing section with tiers and feature breakdowns
  - Refund estimator calculator:
  - Inputs: income, filing status, dependents, deductions
  - Animated estimated refund output

**Document Vault**

- Client login portal
- Drag-and-drop document upload
- AI-style categorization simulation (e.g., W-2s, 1099s, receipts)
- Folder organization and processing status indicators

**RON & Notary Services**

- Service cards describing RON and notary offerings
- “Schedule a Session” booking modal with date/time selector
- Clear process explanation (no “instant booking” language)

**Additional Content**

- Testimonial slider with “Verified Client” badges and refund stories
- Mobile app banner with:
  - Return status tracking preview
  - Direct link to IRS “Where’s My Refund”
- Referral program:
  - Referral code generation (via edge function)
  - Points tracker and 4-tier rewards system
  - Social sharing (Twitter, Facebook, Email)
- Blog / Resource Center:
  - SEO-focused articles
